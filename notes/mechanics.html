Введение в механику

Механика — раздел физики, изучающий движение и равновесие материальных тел под действием сил. Основоположниками классической механики являются Галилео Галилей, Исаак Ньютон и другие ученые.

Основные понятия

  • Материальная точка — тело, размерами которого можно пренебречь в условиях данной задачи
  • Система отсчета — совокупность тела отсчета, системы координат и часов
  • Траектория — линия, вдоль которой движется тело
  • Путь — длина траектории
  • Перемещение — вектор, соединяющий начальное и конечное положение тела

Кинематика

Кинематика изучает движение тел без учета причин, вызывающих это движение.

Основные виды движения:

Прямолинейное равномерное

Движение с постоянной скоростью по прямой линии.

x(t) = x₀ + v·t

Прямолинейное равноускоренное

Движение с постоянным ускорением по прямой линии.

x(t) = x₀ + v₀·t + a·t²/2

Движение по окружности

Тело движется по окружности с постоянной или переменной скоростью.

ω = Δφ/Δt, aц = v²/R

Пример задачи

Условие: Автомобиль движется равноускоренно с начальной скоростью 10 м/с и ускорением 2 м/с². Найдите путь, пройденный за 5 секунд.

Решение:

s = v₀·t + a·t²/2 = 10·5 + 2·25/2 = 50 + 25 = 75 м

Динамика

Динамика изучает причины возникновения движения — силы.

Три закона Ньютона

Первый закон (закон инерции)

Существуют такие системы отсчета, относительно которых тело сохраняет состояние покоя или равномерного прямолинейного движения, если на него не действуют другие тела или действие других тел скомпенсировано.

Второй закон (основной закон динамики)

Ускорение тела прямо пропорционально равнодействующей сил, приложенных к телу, и обратно пропорционально его массе:

a = F/m или F = m·a

Третий закон (закон действия и противодействия)

Силы, с которыми тела действуют друг на друга, равны по модулю и противоположны по направлению:

F₁₂ = -F₂₁

Основные силы в механике:

Сила Формула Направление Примечание
Сила тяжести F = m·g К центру Земли g = 9.81 м/с²
Сила упругости F = -k·x Против деформации Закон Гука
Сила трения F = μ·N Против движения μ - коэффициент трения
Сила Архимеда F = ρ·g·V Вертикально вверх Для жидкостей и газов

Законы сохранения

Импульс

В замкнутой системе векторная сумма импульсов всех тел остается постоянной:

p = m·v
Σpᵢ = const

Механическая энергия

В системе без диссипативных сил полная механическая энергия сохраняется:

E = Eк + Eп = const

Eк = mv²/2, Eп = mgh

Момент импульса

Для замкнутой системы сумма моментов импульсов сохраняется:

L = I·ω = const

где I - момент инерции

Пример: Абсолютно неупругий удар

Условие: Шар массой 2 кг движется со скоростью 3 м/с и сталкивается с неподвижным шаром массой 1 кг. Найти скорость шаров после соударения.

Решение:

m₁v₁ = (m₁ + m₂)u
2·3 = (2 + 1)·u ⇒ u = 2 м/с

Типовые задачи

Задача 1: Движение под углом к горизонту

Условие: Тело брошено под углом 30° к горизонту со скоростью 20 м/с. Найдите время полета и дальность полета.

Решение:

t = 2v₀·sinα/g = 2·20·0.5/9.8 ≈ 2.04 с
L = v₀²·sin(2α)/g = 400·0.866/9.8 ≈ 35.3 м

Задача 2: Наклонная плоскость

Условие: Брусок массой 2 кг скользит по наклонной плоскости с углом 30°. Коэффициент трения 0.1. Найдите ускорение бруска.

Решение:

a = g·(sinα - μ·cosα) = 9.8·(0.5 - 0.1·0.866) ≈ 9.8·0.413 ≈ 4.05 м/с²
