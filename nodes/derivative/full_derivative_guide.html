1. Основные определения и понятия

1.1 Что такое производная?

Производная — фундаментальное понятие математического анализа, которое описывает мгновенную скорость изменения функции. Это ключевой инструмент для анализа динамических систем в физике, экономике, биологии и других науках.

Определение 1.1 (Производная в точке):
Производной функции \( f(x) \) в точке \( x_0 \) называется предел: \[ f'(x_0) = \lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x} \] если этот предел существует и конечен.

Теорема 1.1 (Дифференцируемость и непрерывность):
Если функция \( f(x) \) дифференцируема в точке \( x_0 \), то она непрерывна в этой точке. Обратное, вообще говоря, неверно. \[ f'(x_0) \text{ существует} \Rightarrow f(x) \text{ непрерывна в } x_0 \]
Пример 1.1: Функция \( f(x) = |x| \) непрерывна в точке \( x = 0 \), но не дифференцируема в этой точке, так как: \[ \lim_{\Delta x \to 0^+} \frac{|0 + \Delta x| - |0|}{\Delta x} = 1 \] \[ \lim_{\Delta x \to 0^-} \frac{|0 + \Delta x| - |0|}{\Delta x} = -1 \] Пределы не совпадают, следовательно, производная не существует.

1.2 Геометрическая интерпретация

С геометрической точки зрения производная представляет собой угловой коэффициент касательной к графику функции в заданной точке.

Определение 1.2 (Уравнение касательной):
Уравнение касательной к графику функции \( y = f(x) \) в точке \( (x_0, f(x_0)) \) имеет вид: \[ y = f(x_0) + f'(x_0)(x - x_0) \] где \( f'(x_0) = \tan \alpha \) — угловой коэффициент касательной, равный тангенсу угла наклона.

Масса на пружине

Рассмотрим гармонический осциллятор: масса \( m \) на пружине с жесткостью \( k \). Уравнение движения: \[ m\frac{d^2x}{dt^2} + kx = 0 \] Решение: \( x(t) = A\cos(\omega t) + B\sin(\omega t) \), где \( \omega = \sqrt{\frac{k}{m}} \)

4. Векторный анализ: grad, div, rot

4.1 Градиент

Определение 4.1 (Градиент скалярного поля):
Градиент скалярного поля \( f(\vec{r}) \) — это векторное поле, показывающее направление наибольшего роста функции: \[ \nabla f = \left(\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z}\right) \] где \( \nabla = \left(\frac{\partial}{\partial x}, \frac{\partial}{\partial y}, \frac{\partial}{\partial z}\right) \) — оператор набла.
Визуализация градиента
Активно
Пример 4.1: Рассмотрим скалярное поле \( f(x,y,z) = x^2 + y^2 + z^2 \). Его градиент: \[ \nabla f = \left(2x, 2y, 2z\right) = 2\vec{r} \] Градиент направлен радиально наружу, его модуль равен \( 2r \).

4.2 Дивергенция

Определение 4.2 (Дивергенция векторного поля):
Дивергенция векторного поля \( \vec{F} = (F_x, F_y, F_z) \) — это скалярная величина: \[ \text{div } \vec{F} = \nabla \cdot \vec{F} = \frac{\partial F_x}{\partial x} + \frac{\partial F_y}{\partial y} + \frac{\partial F_z}{\partial z} \] Она характеризует интенсивность источника или стока в точке.
Дивергенция векторного поля
Активно

Теорема Гаусса-Остроградского:
Поток векторного поля через замкнутую поверхность равен интегралу дивергенции по объему: \[ \oint_S \vec{F} \cdot d\vec{S} = \iiint_V (\nabla \cdot \vec{F}) \, dV \]

4.3 Ротор (вихрь)

Определение 4.3 (Ротор векторного поля):
Ротор векторного поля \( \vec{F} = (F_x, F_y, F_z) \) — это векторная величина: \[ \text{rot } \vec{F} = \nabla \times \vec{F} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ \frac{\partial}{\partial x} & \frac{\partial}{\partial y} & \frac{\partial}{\partial z} \\ F_x & F_y & F_z \end{vmatrix} \] или \[ \left(\frac{\partial F_z}{\partial y} - \frac{\partial F_y}{\partial z}, \frac{\partial F_x}{\partial z} - \frac{\partial F_z}{\partial x}, \frac{\partial F_y}{\partial x} - \frac{\partial F_x}{\partial y}\right) \]
Визуализация ротора
Активно

Свойства ротора

  • Характеризует вращение поля
  • Для потенциального поля: \( \nabla \times (\nabla f) = 0 \)
  • Теорема Стокса связывает с циркуляцией
Теорема Стокса:
Циркуляция векторного поля по замкнутому контуру равна потоку ротора через поверхность, натянутую на этот контур: \[ \oint_C \vec{F} \cdot d\vec{l} = \iint_S (\nabla \times \vec{F}) \cdot d\vec{S} \]

5. Численные методы дифференцирования

5.1 Конечные разности

Методы численного дифференцирования:
  • Правая разность: \( f'(x) \approx \frac{f(x+h)-f(x)}{h} \)
  • Левая разность: \( f'(x) \approx \frac{f(x)-f(x-h)}{h} \)
  • Центральная разность: \( f'(x) \approx \frac{f(x+h)-f(x-h)}{2h} \) (более точная)
  • Пять точек: \( f'(x) \approx \frac{-f(x+2h)+8f(x+h)-8f(x-h)+f(x-2h)}{12h} \)

5.2 Автоматическое дифференцирование


class DualNumber {
    constructor(value, derivative = 0) {
        this.value = value;
        this.derivative = derivative;
    }
    
    add(other) {
        return new DualNumber(
            this.value + other.value,
            this.derivative + other.derivative
        );
    }
    
    multiply(other) {
        return new DualNumber(
            this.value * other.value,
            this.derivative * other.value + this.value * other.derivative
        );
    }
    
    sin() {
        return new DualNumber(
            Math.sin(this.value),
            Math.cos(this.value) * this.derivative
        );
    }
}

// Пример использования
const x = new DualNumber(Math.PI/2, 1); // x = π/2, x' = 1
const y = x.sin();
console.log(`sin(π/2) = ${y.value}, производная = ${y.derivative}`);
// Вывод: sin(π/2) = 1, производная = 0
